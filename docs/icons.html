UI Icons

Google Material Symbols icon set with semantic labels for consistent interface iconography.

Usage

Include the Material Symbols font via CDN, then use the icon name inside a <span class="material-symbols-outlined"> element. Click any icon below to copy its markup.

PropertyValue

Default size24px (optical size 24)
Weight300 (regular)
Fill0 (outlined)
