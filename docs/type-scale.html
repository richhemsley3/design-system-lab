Usage Guidelines

Semantic sizing — Use the semantic token name that matches the content's role, not its visual size:
  • Page titles: --sds-type-display with --sds-weight-semibold
  • Section headings: --sds-type-heading with --sds-weight-light
  • Subsections & subtitles: --sds-type-subheading with --sds-weight-semibold
  • Card titles, large labels: --sds-type-title with --sds-weight-semibold
  • Body copy, button labels: --sds-type-body with --sds-weight-regular
  • Nav links, table cells: --sds-type-label with --sds-weight-regular
  • Chart labels, metadata: --sds-type-caption with --sds-weight-regular
  • Demo labels, overlines: --sds-type-overline with --sds-weight-semibold + uppercase

Pair --sds-leading-tight (1.2) with headings, --sds-leading-normal (1.5) with body text, and --sds-leading-relaxed (1.6) with long-form content. Use --sds-tracking-tight for display sizes and --sds-tracking-wide for uppercase overlines.
