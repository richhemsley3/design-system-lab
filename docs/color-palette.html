Color Palette

Complete token reference for all color scales, semantic mappings, and usage guidelines.

Cool Gray Dark mode only

Neutral cool-toned grays for dark mode backgrounds, surfaces, and text elements.

Usage guidance: Use cool gray tokens in dark mode to create visually consistent and accessible interfaces. Avoid cool gray in light mode — it clashes with warm gray surface colors.

Warm Gray Light mode

Warm neutral grays for light mode backgrounds, borders, and text. The primary neutral palette.

Blue

Primary brand and interactive color. Used for buttons, links, active navigation, and focus states.

Red

Error, risk, and destructive action states. Used for alerts, validation errors, and risk badges.

Green

Success and positive status. Used for "Great", "Good", approvals, and protection indicators.

Yellow

Warning and moderate states. Used for "Fair", "Moderate", caution alerts, and pending items.

Purple

Accent color for categorization and decorative use.
