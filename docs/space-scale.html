Usage Guidelines

Base unit: 8px — The scale is anchored around multiples of 8px for larger values and 4px for fine control. Common mappings:
  • Inline spacing (icon-to-label gaps): --sds-space-150 (6px) or --sds-space-200 (8px)
  • Component padding: --sds-space-200 (8px) to --sds-space-400 (16px)
  • Section gaps: --sds-space-600 (24px) to --sds-space-800 (32px)
  • Page-level margins: --sds-space-1000 (40px) to --sds-space-1200 (48px)

Do not tokenize: font sizes, border widths, border-radius, element dimensions (button heights, icon sizes), or values that don't align with the scale.
