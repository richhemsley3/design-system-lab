Form Inputs

Text fields, selects, checkboxes, radios, and toggles for building enterprise forms. All inputs share consistent sizing, spacing, and state patterns.

Text Inputs

Single-line text fields used for capturing short-form data. The base class .sds-input provides consistent styling across all states and sizes. Wrap in .sds-field to pair with a label and helper text.

States

All interactive states for the text input. Focus uses a 2px blue border with a subtle outer glow. Error replaces the border with red and displays validation messaging below.
