Search

Search input for finding projects, settings, activity logs, and other resources. Supports suggestions, filter chips, and keyboard navigation.

Basic Search

Input with a leading search icon and clear button. Focus ring uses a 2px border-focus border with a 3px rgba(1,61,91,0.15) box-shadow.
