Tables

Data tables for displaying, sorting, and acting on structured information. Three progressive patterns from static display to full interactive management.

Combines sorting, pagination, and bulk actions. Select rows with checkboxes to reveal a contextual action bar. The header checkbox toggles all visible rows and shows an indeterminate state for partial selections.
