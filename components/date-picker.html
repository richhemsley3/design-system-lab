Date Picker

Calendar-based date selection for log filtering, scheduling, date ranges, and report periods in enterprise workflows.

Single Date Picker

An input field with a calendar icon that opens a dropdown calendar on click. Navigate months with arrow controls and select a single date. Today's date is highlighted, and the selected date uses the primary accent color.

Default
Closed
Open
Interactive

Click the input to open, select a date, and use arrow keys to navigate.

Date Range Picker

Two connected inputs for selecting a start and end date. The calendar highlights the selected range. Quick-select presets provide common date ranges used in security audit workflows.

Calendar Grid

A standalone inline calendar, always visible. Useful for dashboard widgets or embedded date selection where a dropdown pattern is not needed.

Inline calendar

With Time

Date and time picker for scheduling policies, setting scan times, or defining precise audit windows. The time input sits below the calendar within the same panel.

Date + Time

Presets Panel

A side panel with quick-select options displayed alongside the calendar. Ideal for report period selection where users frequently choose standard intervals.
