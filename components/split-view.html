Vertical Split

Top and bottom panes separated by a horizontal divider. Useful for query editors with results panels below.

Vertical — Query / Results

SELECT
  u.email,
  u.full_name,
  u.department,
  COUNT(p.id) AS project_count
FROM users u
LEFT JOIN projects p
  ON p.owner_id = u.id
  AND p.status = 'active'
WHERE u.role = 'manager'
GROUP BY u.id
ORDER BY project_count DESC
LIMIT 25;
