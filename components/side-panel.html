Side Panel

Slide-in panels and drawers for viewing details, editing properties, and secondary navigation. Side panels keep the user in context while revealing additional content alongside the main view.

Basic Side Panel

Slides in from the right edge with an overlay backdrop. Contains a header with title and close button, a scrollable body area, and an optional sticky footer with action buttons. Available in three width sizes.

Resource Properties

Use the narrow panel for simple property editors, quick previews, or notification lists that don't require much horizontal space.

The panel scrolls vertically when content exceeds the available height, keeping the header and footer fixed in place.

Without Overlay

The push variant slides in without a backdrop overlay. Ideal for persistent detail views or inspector panels where the user should still interact with the main content area. No click-outside dismissal.
