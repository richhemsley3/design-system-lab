Alerts & Banners

Inline alerts, page banners, and compact feedback messages for communicating status, warnings, errors, and informational notices across the application.

Inline Alerts

Full-width alerts that appear within the page flow. Each variant uses a tinted background, colored border, and icon badge to communicate severity. Use for persistent messages that require user attention.

All Variants
info
System update available
A new version of the automation engine is available. Review the changelog before upgrading.
check_circle
Changes published
Your changes have been published and are now active across all environments.
warning
Approaching storage limit
You have used 85% of your storage quota. Consider archiving older records to avoid disruption.
error
Connection failed
Unable to reach the database server. Check your network settings and try again.

Alerts with Actions

Alerts can include action links or buttons to let users respond directly from the notification. Use a primary action for the recommended path and an optional secondary action for alternatives.

With Action Links
warning
Deadline approaching
Your quarterly review is due in 7 days. Complete the assessment to stay on track.
error
3 issues detected
Critical issues were found during the latest check of your connected services.
info
New integration available
Connect your database to enable automated syncing and rule enforcement.

Dismissible Alerts

Interactive close button that removes the alert with a smooth fade-out animation. Use for messages that can be acknowledged and cleared by the user.

Click the close icon to dismiss
info Tip: You can drag columns to reorder the table layout.

warning
Token expiring soon
Your API access token expires in 48 hours. Regenerate it to maintain uninterrupted service.
error
Sync failed
The scheduled sync could not complete. 12 items were skipped due to permission errors.

Page Banners

Full-width persistent banners that sit at the top of the page for system-level messages such as scheduled maintenance, trial expiry notices, or compliance deadlines. They span edge to edge with no border-radius.

System-level Banners
info
Scheduled maintenance: The platform will be unavailable on March 20 from 2:00 AM to 4:00 AM UTC.
warning
Trial expiring: Your enterprise trial ends in 5 days. Upgrade now to retain all data and configurations.
error
Overdue: Your quarterly review is past the deadline. Immediate action required.
check_circle
Migration complete: All datasets have been successfully migrated to the new data center region.
Page content below banners

Compact Alerts

Smaller inline alerts for form-level or section-level feedback. Reduced padding and icon size for tighter layouts, such as validation messages inside forms or contextual hints within settings panels.

Form-level Feedback
error Connection string is invalid. Check the host and port values.
check_circle Connection test passed. Database is reachable.
warning This name already exists. Saving will overwrite the existing version.
info Rules are evaluated top-to-bottom. Drag to reorder priority.
Compact with Title & Dismiss
