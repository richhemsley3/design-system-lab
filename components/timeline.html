Timeline / Activity Log

Vertical timeline for activity feeds, change history, and event logs. Essential for enterprise products that need to show chronological event sequences.

Basic Timeline

Simple vertical timeline with dot markers connected by a line. Events are listed in reverse-chronological order with timestamps.
