Pagination

Navigation controls for moving through paged data sets. Six variants from full page-number controls to infinite scroll, designed for enterprise data security workflows.

Basic Pagination

Page number buttons with previous/next arrows and ellipsis for large page ranges. The active page is highlighted with the sds-pagination__btn--active class. Use aria-current="page" on the active button for accessibility.

Load More

Button-based pagination that appends additional results below the current set. Shows a count indicator so users know how many items have been loaded and how many remain. Suitable for feeds, activity logs, or audit trails.

Cursor / Infinite Scroll

Scroll-based pagination that automatically loads more content as the user scrolls toward the bottom. Shows a spinner and status text to indicate loading state. Useful for continuous feeds like security event streams or data discovery results.
