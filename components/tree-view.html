Tree View

Selectable Tree

Tree with checkbox selection. Selecting a parent node selects all of its children. Partial child selection shows an indeterminate state on the parent. Uses the canonical .sds-tree-checkbox pattern.

CSS Classes

ClassDescription
.sds-treeRoot container for the tree

.sds-tree-item__rowClickable row content wrapper

.sds-tree-item__toggle--hiddenInvisible placeholder toggle for leaf nodes (preserves alignment)
.sds-tree-item__iconFile or folder icon area
.sds-tree-item__labelText label for the node
.sds-tree-item__metaRight-aligned metadata slot (counts, tags, types)
.sds-tree-item__tagInline status tag within meta
.sds-tree-groupContainer for nested children, provides depth indentation
.sds-tree-item--selectedModifier: highlighted selected state
.sds-tree-item--disabledModifier: non-interactive disabled state
.sds-tree-checkboxCheckbox wrapper for selectable tree variant; uses canonical .sds-checkbox-group-box from selection-controls
