Breadcrumbs

Hierarchical navigation showing the user's current location within the application structure. Essential for deep enterprise workflows like settings management, resource browsing, and activity logs.

Basic Breadcrumbs

Simple text links separated by chevron icons. The last item represents the current page and is rendered as bold, non-interactive text with aria-current="page".

Collapsed / Truncated

When breadcrumb trails exceed four levels, middle items are collapsed behind an ellipsis button. Clicking the ellipsis expands the full path inline. This keeps the component compact while still providing access to the full hierarchy.
