Basic Code Block

A simple code block with monospace font, dark background, and rounded corners. Used for displaying database queries, configuration values, and other code content.

SQL query
SELECT name, email, role FROM team_members WHERE department = 'engineering' LIMIT 100;

With Header

Code block with a header bar showing the language label and an interactive copy-to-clipboard button. Clicking the button copies content and briefly shows a confirmation tooltip.

Header with language label and copy action
SQL 
SELECT
  u.email,
  u.full_name,
  p.project_name,
  p.last_updated
FROM users u
JOIN project_assignments p ON u.user_id = p.user_id
WHERE p.status = 'active'
  AND p.last_updated > '2025-01-01'
ORDER BY p.last_updated DESC
LIMIT 50;
