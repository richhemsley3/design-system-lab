Status Indicator

Live connection and health status indicators showing real-time system state. Different from tags and badges because these represent live, dynamic state such as connected/disconnected, syncing/idle, and healthy/degraded.

All States

Each state maps to a specific dot color. Active states pulse to convey live activity.

Always pair the status dot with a text label. Color alone should not be the sole indicator of state.
