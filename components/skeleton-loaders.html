Skeleton Loaders

Placeholder UI that mimics content layout while data is loading. Reduces perceived wait time and prevents layout shift in the data security dashboard.

Token reference for building and customising skeleton loaders.

Design Tokens

PropertyToken / ValueNotes
Base color --sds-color-warm-gray-100 Resting skeleton surface
Highlight color --sds-color-warm-gray-150 Shimmer peak
Animation sds-skeleton-shimmer 1.5s ease-in-out infinite Left-to-right gradient sweep
Text line height 14px Matches body text line height
Border radius — text 4px Rounded pill for text lines
Border radius — rect 6px Matches card image radius
Border radius — circle 50% Avatar placeholder
Line gap --sds-space-200 Space between text skeleton lines
