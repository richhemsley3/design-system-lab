Popover

Click-triggered floating panels for contextual content, confirmations, quick actions, and inline forms. Popovers provide richer interactions than tooltips without the weight of a full modal.

Basic Popover

Click-triggered floating panels with a title, body text, and optional action links. An arrow caret points to the trigger element. White background with a subtle shadow and 8px border-radius. Supports four placement positions: top, bottom, left, and right.

Confirmation Popover

A lightweight inline confirmation that appears as a popover instead of a full modal dialog. Best suited for simple destructive actions where you want to confirm intent without interrupting the user's flow.

Click to trigger confirmation

Delete masking rule?

Revoke API key?

Any services using this key will immediately lose access. You can generate a new key afterward.

Publish policy changes?

This will apply the updated access control policy across all production data sources immediately.
