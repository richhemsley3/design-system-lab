Numeric badges, overlay dots, and inline status indicators for communicating counts, presence, and security severity across the data platform.

Severity Badges

Color-coded labels for categorizing security findings by severity level. Each level uses a distinct color to ensure quick identification in security dashboards and reports.

All severity levels
Critical High Medium Low Info

In Context

Severity badges as they appear alongside security finding descriptions.
