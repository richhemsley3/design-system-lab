Segmented Control

A compact toggle for switching between 2–4 mutually exclusive options. More compact than tabs, used for view toggles, time range selectors, and mode switches.

Basic

Two-option segmented control for toggling between views. Click to switch the active state.

Interactive — two options

Three Options

Time range selector with three segments. Ideal for dashboard filters.

Interactive — three options

Four Options

View mode filter with four segments. Use sparingly — four is the recommended maximum.

Interactive — four options

With Icons

Options can include a leading Material Symbols icon using .sds-segmented__icon. An icon-only variant omits text labels for compact toolbars.

Icon + text
Icon only

Sizes

Three sizes available: small (.sds-segmented--sm), default, and large (.sds-segmented--lg).

Block (Full-Width)

Use .sds-segmented--block to stretch the control to fill its container width, with each option sharing equal space.
