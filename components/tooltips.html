Tooltips & Popovers

Contextual overlays that provide additional information on hover or click. Use tooltips for brief hints and popovers for richer, interactive content.

Simple Tooltips

Dark, compact overlays that appear on hover after a 150ms delay. Use for short descriptions, keyboard shortcut hints, or label clarification. Supports four placement positions.

Hover each button

Rich Tooltips

Tooltips with a title and description for when a short label is not enough. Wider max-width and increased padding provide room for two-line explanations.

Role-based access

Popovers

Light-background overlays triggered by click. Popovers can contain structured content, links, and action buttons. Dismiss with the close button or by clicking outside.

Info Icon Pattern

A common enterprise pattern: a small info icon placed beside a field label. On hover, it reveals a tooltip explaining the setting or concept in plain language.

Hover the info icons
Data Classification info Determines encryption level, access controls, and retention policies applied to this dataset.
Masking Policy info Controls how sensitive fields are redacted when viewed by users without elevated permissions.
Replication Factor info Number of redundant copies stored across availability zones. Higher values improve durability.
Row-Level Security info Restricts query results so users only see rows they are authorized to access based on their role.

Data Source Configuration

Connection Name info A human-readable name for this data source. Used in audit logs and alert notifications.
Encryption Standard info Encryption algorithm applied to data at rest. AES-256 is recommended for compliance with SOC 2 and HIPAA.
Retention Period info How long data is kept before automatic deletion. Must meet your organization's regulatory requirements.
Access Control info Determines which roles can query this data source. Restricted sources require explicit grant from a Security Officer.
