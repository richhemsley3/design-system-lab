Keyboard Interactions

Key Action Context
⌘K Open command palette Global — works anywhere in the application
Escape Close command palette While palette is open
↑ / ↓ Navigate between result items While palette is open, wraps around
Enter Select the active result Activates the highlighted item

ARIA Roles

Element Role / Attribute Notes
Overlay role="dialog", aria-label Announced as dialog on open
Palette container role="combobox", aria-expanded, aria-haspopup="listbox" Composite widget pattern
Results list role="listbox", aria-label Contains all option items
Result item role="option" Each selectable result
Search input aria-activedescendant Points to active option ID
